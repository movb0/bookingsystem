# 🏢 Hall Booking Management System

A Java Swing desktop application connected to a REST API for managing hall bookings.  
The system supports authentication, hall management, and booking functionality using a backend API.

---

## 🚀 Features

- 🔐 User Login with JWT Authentication
- 🏛️ Add / View Halls
- 📋 Display halls in JTable
- 💾 Save hall data to backend database via API
- 🔄 Load real-time data from API
- 🎨 Clean Swing UI design
- 🔑 Secure API communication using Bearer Token

---

## 🛠️ Technologies Used

- Java (JDK 8+)
- Java Swing (GUI)
- REST API (HTTPURLConnection)
- JSON Parsing (org.json)
- IntelliJ IDEA
